# AFMC Mess App (Basic Starter)

Basic starter structure for AFMC Mess Management Web Application.

## Stack
- React.js (client)
- Express.js (server)
- MySQL (database)

## Notes
This ZIP contains the **project structure + starter placeholder files** only.
You can now start implementing modules incrementally:
1. Authentication
2. Inventory
3. Menu & Orders
4. Payments
5. Reports
